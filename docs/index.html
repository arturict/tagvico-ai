Quick Install

Up and running in under two minutes. No external database required.

⚙ Docker Compose

Save as docker-compose.yml and bring up the stack:

services:
  tagvico-ai:
    image: ghcr.io/arturict/tagvico-ai:latest
    container_name: tagvico-ai
    restart: unless-stopped
    ports:
      - "8080:3000"
    environment:
      PAPERLESS_API_URL: http://paperless-ngx:8000
      AI_PROVIDER: openai
      OPENAI_API_KEY: ${OPENAI_API_KEY}
      OPENAI_MODEL: gpt-5.4-mini
    volumes:
      - tagvico_ai_data:/app/data

volumes:
  tagvico_ai_data:

▶ One-line Start

Prefer a single command? Run the image directly:

docker run -d \
  --name tagvico-ai \
  -p 8080:3000 \
  -e PAPERLESS_API_URL=http://paperless:8000 \
  -e AI_PROVIDER=openai \
  -e OPENAI_API_KEY=sk-... \
  -v tagvico_ai_data:/app/data \
  ghcr.io/arturict/tagvico-ai:latest

Then open http://localhost:8080/setup to finish onboarding. For Ollama, OpenRouter, LM Studio, and Azure providers, see the full provider docs.

How It Works

Three steps from install to fully filed documents.

1
🔗

Connect Paperless

Point Tagvico at your Paperless-ngx instance and paste an API token. The setup wizard auto-detects your installation.

2
⚙

Choose Provider

Pick OpenAI, OpenRouter, Ollama, LM Studio, or Azure. Keep documents fully local with Ollama or send to the cloud — your call.

3
✓

Review Results

Tagvico polls for new documents, generates titles, tags, correspondents, types, and owners, then writes them back via the Paperless API.
